# BeInside 타겟 페르소나 요약

콘텐츠를 작성하기 전에 이 페르소나들을 머릿속에 떠올리세요.
모든 콘텐츠는 이 네 명 중 한 명 이상을 위해 존재합니다.

---

## P1 이수진 (34세) — 한부모 엄마

| 항목 | 내용 |
|------|------|
| 상황 | 혼자 아이를 키우는 한부모. 직장과 육아를 동시에 감당 중. |
| 핵심 니즈 | 핵심만 5분 이내에 파악. 긴 글 읽을 시간 없음. |
| 기기 | 스마트폰 90%, PC 10% |
| 고통 포인트 | 정서적 고립. "내가 잘 하고 있는 건가?" 불안. 판단받는 것에 대한 두려움. |
| 콘텐츠 전략 | 핵심 먼저. 3줄 이내 요약. 판단 없는 어조. 실행 가능한 팁. |

---

## P2 박지연 (28세) — 혼자 출산·산후 회복 중

| 항목 | 내용 |
|------|------|
| 상황 | 파트너 없이 또는 충분한 지원 없이 출산·산후 회복 중. |
| 핵심 니즈 | "이 증상 정상인가요?" 즉각 확인. 불안 감소. |
| 기기 | 스마트폰 100% (몸을 움직이기 어려운 상황에서 사용) |
| 고통 포인트 | 고립감. 새벽에 혼자 증상 검색하는 공포. 의료 정보 신뢰 부족. |
| 콘텐츠 전략 | "이건 정상이에요" / "이건 병원 가세요" 이분법 명확하게. 야간 접근성 고려. |

## P3 김민준 (16세) — 감정을 혼자 삭이는 청소년

| 항목 | 내용 |
|------|------|
| 상황 | 학교·가정에서 힘들지만 어른에게 말 못 함. 자기 전 혼자 검색. |
| 핵심 니즈 | 판단 없는 정보. 어른에게 안 들키는 콘텐츠. "나만 이런 게 아니구나." |
| 기기 | 스마트폰 100%, 주로 자기 전 11pm~1am |
| 고통 포인트 | "말하면 더 힘들어질 것 같아서" 침묵. 낙인 두려움. |
| 콘텐츠 전략 | 청소년 톤(반말/해요체 혼용). 이모지. 짧은 문장. "너는 혼자가 아니야" 먼저. |

## P4 최준호 (41세) — 초보 아빠

| 항목 | 내용 |
|------|------|
| 상황 | 첫 아이를 키우는 아빠. 육아에서 소외되는 느낌. 어디서 물어봐야 할지 모름. |
| 핵심 니즈 | 5~10분 요약. 아빠 관점 정보. "내가 할 수 있는 게 뭔가요?" |
| 기기 | PC 40%, 스마트폰 60% |
| 고통 포인트 | 육아 정보가 대부분 엄마 중심. "아빠는 뭘 해야 하나" 막막함. |
| 콘텐츠 전략 | 아빠 역할 명시. 실행 가능한 액션 아이템. PC에서도 읽기 좋은 구조. |

---

## 공통 인사이트 (모든 스킬의 근본 원칙)

1. **정보가 아닌 "기준"이 필요하다** — "지금 이게 정상인가요?"라는 불안에 답하는 것이 우선
2. **"혼자"라는 감각이 공통 고통이다** — 정서적 고립 해소가 핵심 가치
3. **모바일 first, 짧게, 지금 당장** — 길게 읽을 여유가 없는 상황에서 접근
4. **공식 채널에 대한 불신과 낙인 두려움** — 낮은 문턱의 첫 접점이 되어야 함
