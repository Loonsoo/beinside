## 출처 작성 형식

데이터.js 내 `stat.label` 필드에 기입할 때:

```
"[수치] — [설명] ([출처명] [연도])"

예: "27.7% — 중·고등학생 우울감 경험률 (여성가족부 2025 청소년 통계, 2024년 기준)"
```

---

## 새 출처 추가 절차

1. 이 파일에 새 행을 추가 (ID는 S13, S14... 순서)
2. `index.html` > `#source-drawer` 섹션에 출처 항목 추가
3. `js/data.js`의 해당 `stat.label`에 출처 ID와 함께 기재
4. 국제 출처: 원문 연도 + "국제 연구" 명시
5. 업데이트 예상 시점이 지났다면 팀에 갱신 요청
