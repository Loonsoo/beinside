
## 그림자 시스템

```css
/* 기본 카드 */
box-shadow: 0 2px 12px rgba(0,0,0,.06);
/* 카드 hover */
box-shadow: 0 8px 24px rgba(0,0,0,.10);
/* 플로팅 버튼 */
box-shadow: 0 4px 16px rgba(0,0,0,.18);
/* focus 링 */
box-shadow: 0 0 0 3px var(--peach-glow);
```
