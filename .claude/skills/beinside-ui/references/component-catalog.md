# BeInside 컴포넌트 카탈로그

현재 `css/styles.css`와 `index.html`에 존재하는 모든 UI 컴포넌트를 분류한 문서입니다.
새 컴포넌트를 추가하기 전에 여기서 재사용 가능한 것이 있는지 먼저 확인하세요.

---

## 카드

| 클래스 | 주요 속성 | 사용 위치 |
|--------|-----------|-----------|
| `.card` | `border-radius: 18px`, 기본 그림자, 호버 시 `translateY(-4px)` | 전체 콘텐츠 카드 |
| `.card.full` | 전체 너비 (그리드 colspan) | 강조 콘텐츠 카드 |
| `.card.cbg-a` | 살구 그라데이션 배경 | 주요 정보 강조 |
| `.card.cbg-s` | 세이지 그린 배경 | 성장 관련 |
| `.card.cbg-p` | 퍼플 배경 | 감정·정신 관련 |
| `.card.cbg-sky` | 스카이 배경 | 안내·정보 |
| `.card.cbg-lavender` | 라벤더 배경 | 차분한 내용 |
| `.card.cbg-mint` | 민트 배경 | 건강·긍정 |
| `.card.cbg-rose` | 로즈 배경 | 가족·사랑 |
| `.card.cbg-teal` | 틸 배경 | 전문·신뢰 |
| `.alone-card` | 혼자인 아이/청소년 지원 카드 특수 스타일 | 혼자인 사람 섹션 |

## 버튼

| 클래스 | 주요 속성 | 사용 위치 |
|--------|-----------|-----------|
| `.gobtn` | 메인 CTA 버튼, `min-height: 44px`, 피치 배경 | 주요 행동 유도 |
| `.pill` | 알약형 버튼, `border-radius: 20px` | 태그 선택, 필터 |
| `.sit-btn` | 상황별 가이드 진입 버튼 | 상황 선택 화면 |
| `.sp-age-tab` | 상황별 연령 탭 버튼 | 상황 가이드 내 탭 |
| `.birth-tab` | 출산 단계 탭 버튼 | 출산 가이드 탭 |
| `.view-toggle button` | 목록/그리드 뷰 전환 버튼 | 헤더 내 |
| `.quick-exit-btn` | 빠른 탈출 버튼 | 가정폭력 페이지 전용 |

## 입력

| 클래스 | 주요 속성 | 사용 위치 |
|--------|-----------|-----------|
| `.ninput` | 월령 숫자 입력 | 월령 검색 |
| `.form-input` | 일반 텍스트 입력, focus 시 피치 보더 + 글로우 | 프로필 생성 폼 |
| `.memo-textarea` | 메모 텍스트 영역, 리사이즈 가능 | 메모 입력 |

## 레이아웃

| 클래스 | 주요 속성 | 사용 위치 |
|--------|-----------|-----------|
| `.grid` | `display: grid`, 2열 (데스크톱) → 1열 (모바일) | 메인 콘텐츠 그리드 |
| `.tgrid` | 3열 그리드 | 팁·항목 3열 배열 |
| `.fgrid` | 4열 그리드 | 발달 이정표 등 |
| `.hgrid` | 수평 스크롤 그리드 | 가로 스크롤 카드 열 |
| `.sp-grid` | 상황별 가이드 전용 그리드 | 상황 가이드 레이아웃 |
| `.main-col` | 전체 너비 메인 영역 | 사이드바 제거 후 메인 |
| `section.alt` | `background: var(--section-alt)` | 교차 배경 섹션 |

## 내비게이션

| 클래스/요소 | 주요 속성 | 사용 위치 |
|------------|-----------|-----------|
| `header` | `sticky top: 0`, 64px 고정 높이, `backdrop-filter: blur(28px)` | 전체 상단 헤더 |
| `header nav` | 상단 내비 링크 | 헤더 내 |
| `.view-toggle` | 뷰 전환 토글 (목록/그리드) | 헤더 우측 |
| `.mobile-tabbar` | 하단 탭바, 60px + `safe-area-inset-bottom` | 640px 이하 |
| `#scroll-bar` | 상단 스크롤 프로그레스 바, 3px 높이 | 전역 |
