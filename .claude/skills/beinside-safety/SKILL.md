# BeInside 안전 콘텐츠 스킬

## 최우선 원칙

**사용자의 안전 > 정보의 정확성 > 콘텐츠의 완성도**

콘텐츠가 불완전하더라도, 사용자가 위험에 노출되는 것보다 낫습니다.
의심스러우면 더 안전한 방향을 선택하세요.

## 빠른 탈출 버튼 사양

가정폭력 관련 **모든 페이지**에 필수 배치합니다.

### HTML
```html
<button
  class="quick-exit-btn"
  onclick="quickExit()"
  aria-label="페이지 나가기"
>✕</button>
```

### CSS
```css
.quick-exit-btn {
  position: fixed;
  top: 12px; right: 12px;
  z-index: 9999;
  min-width: 44px; min-height: 44px;
  border-radius: 50%;
  background: rgba(0,0,0,.7);
  color: white;
  font-size: 18px;
  border: none;
  cursor: pointer;
}
```

### JavaScript
```js
function quickExit() {
  // 현재 페이지 기록을 네이버로 대체 (뒤로 가기 방지)
  history.replaceState(null, '', 'https://www.naver.com');
  // 네이버로 이동
  location.href = 'https://www.naver.com';
}
```

### 설계 원칙
- 라벨: "✕" 또는 "나가기" — 가해자가 봐도 모호하게
- 위치: 화면 우상단 고정, `z-index` 최상위
- 크기: 모바일에서 44px × 44px 이상 (손가락으로 쉽게 터치)
- `history.replaceState()`로 기록 대체 — 뒤로 가도 BeInside로 돌아오지 않음

## 긴급 연락처 규칙

`references/helpline-database.md`를 항상 참조합니다.

- 모든 번호에 **한 줄 설명** 필수: 무료 여부 / 24시간 여부 / 전화하면 어떻게 되는지
- 비밀보장·기록 정책을 **단정 짓지 않습니다**. "자세한 내용은 전화 시 확인해 주세요." 추가.
- 전화번호는 반드시 `<a href="tel:번호">` 링크 형태

```html
<a href="tel:1393" class="emergency-call-btn">
  📞 1393 자살예방상담전화 · 무료 · 24시간
</a>
```

## 가정폭력 콘텐츠 규칙

`references/domestic-violence-guide.md` 참조.

### 절대 금지
- "왜 떠나지 않나요?" 류의 표현 — 피해자 탓 금지
- "~하면 됩니다" — 상황이 단순하지 않음을 인식
- 피해자를 비교하거나 판단하는 표현

### 필수
- 체크리스트·실행 가능한 형태 우선
- 가해자가 화면을 볼 수 있음을 전제로 UI 설계
- 빠른 탈출 버튼 배치 (이 스킬의 사양대로)
- 안전 계획 수립 가이드 포함

## 자살·자해 예방 콘텐츠 규칙

`references/suicide-prevention-media-guide.md` 참조.

### 절대 금지
- 자살 방법 구체적 기술
- 특정 장소·수단 언급
- "극단적 선택"이라는 완곡어 — 오히려 직접적 표현("자살")이 도움이 됨

### 필수
- 위기 상담 연결을 **콘텐츠 상단**에 배치
- "자살은 예방 가능하며 도움이 존재한다"는 메시지 포함
- 1393 자살예방상담전화 반드시 포함

### 권장 표현
- "지금 많이 힘드시죠. 혼자 감당하지 않아도 돼요."
- "전화 한 통이 오늘을 바꿀 수 있어요."
- "도움을 요청하는 것은 용기 있는 행동이에요."

## 아동학대 콘텐츠 규칙

- 신고 의무자 정보 명시 (교사, 의료인 등)
- 112 신고 절차를 구체적으로 안내
- 피해 아동의 관점에서 작성 (죄책감 제거)
- 부모/보호자 대상 콘텐츠: 판단 없이 "도움을 받을 수 있다"는 메시지

## 안전 콘텐츠 완료 후 체크리스트

- [ ] 빠른 탈출 버튼이 페이지에 포함되어 있는가 (가정폭력 주제)
- [ ] 위기 상담 연락처가 콘텐츠 상단에 있는가
- [ ] 피해자 탓 표현이 없는가
- [ ] 자살 방법·장소·수단을 구체적으로 언급하지 않았는가
- [ ] 모든 전화번호가 tel: 링크 형태인가
- [ ] 비밀보장 관련 단정적 표현이 없는가
- [ ] "자세한 내용은 전화 시 확인해 주세요" 문구가 있는가
