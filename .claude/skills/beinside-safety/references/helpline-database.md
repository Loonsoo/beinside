## 콘텐츠 작성 시 표준 형식

```
📞 [번호] [이름]
무료 | [24시간 여부]
→ [전화하면 어떻게 되는지 한 줄]
자세한 내용(비밀보장 범위, 기록 정책)은 전화 시 확인해 주세요.
```

**예시:**
```
📞 1393 자살예방상담전화
무료 | 24시간
→ 상담사가 전화를 받아 지금 힘든 감정을 함께 듣고 대화해요.
자세한 내용(비밀보장 범위, 기록 정책)은 전화 시 확인해 주세요.
```
