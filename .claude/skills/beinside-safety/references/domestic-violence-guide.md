# 가정폭력 콘텐츠 작성 가이드

가정폭력 피해자는 극도로 취약한 상황에 있습니다.
콘텐츠는 피해자에게 힘이 되어야 하며, 결코 추가적 상처를 주어서는 안 됩니다.

---

## 절대 금지 표현

| 금지 표현 | 이유 |
|-----------|------|
| "왜 떠나지 않나요?" | 피해자를 탓하는 구조. 떠나는 것이 불가능한 이유가 수십 가지 있음. |
| "~하면 됩니다" | 상황이 단순하지 않음. 해결책을 단정하는 것이 오히려 무력감 강화. |
| "관계를 개선하세요" | 폭력은 관계 문제가 아닌 범죄. 피해자가 관계를 개선할 책임이 없음. |
| "참고 견디면..." | 폭력은 시간이 지나도 멈추지 않음. 인내를 권장하는 표현 금지. |
| "가해자도 힘든 상황이라서" | 가해 행동을 정당화하는 표현 절대 금지. |
| 피해자끼리 비교 | "다른 분들은 잘 빠져나왔는데..." 류 금지. |

---

## 필수 포함 요소

### 1. 안전 우선 메시지
콘텐츠 상단에 반드시:
```
지금 위험하다면 112에 전화하세요.
당신의 안전이 가장 중요합니다.
```

### 2. 빠른 탈출 버튼
SKILL.md의 사양대로 구현 (가해자가 화면을 볼 수 있다는 전제).

### 3. 실행 가능한 형태
- 체크리스트: "지금 할 수 있는 것"
- 단계별 안내: "오늘 밤 안전한 장소가 없다면 → 내일 준비할 수 있다면 → 장기적으로..."

### 4. 피해자의 감정 인정
- "당신이 겪고 있는 것은 당신 탓이 아닙니다."
- "폭력에는 이유가 없습니다."
- "도움을 받을 자격이 있습니다."

---

## UI 설계 원칙 (가해자가 볼 수 있음을 전제)

- 페이지 제목: 최대한 모호하게 (예: "가족 지원 가이드")
- 빠른 탈출 버튼: 항상 화면에 표시
- 브라우저 기록 대체: `history.replaceState()` 사용
- 사진·이미지: 폭력 장면 절대 사용 금지

---

## 가정폭력 유형별 콘텐츠 지침

### 신체 폭력
- 응급 시 112 우선
- 증거 기록 방법 안내 (사진, 의료 기록)
- 의료 지원 연결

### 정서적 폭력 (언어폭력, 통제)
- "이것도 폭력입니다" 명시
- 정서적 폭력의 징후 체크리스트
- 심리 상담 연결 (1366)

### 경제적 폭력
- "경제적 통제도 폭력입니다" 명시
- 긴급 지원 자원 안내
- 법적 권리 정보 (1644-6621 한부모가족지원센터)

### 아동이 있을 경우
- 아이를 위한 안전 계획 포함
- 아동학대 가능성 안내 (1899-3075)
- "아이를 위해 참는 것"이 항상 최선이 아님을 설명
