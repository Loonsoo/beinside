# BeInside 언어 우선순위 가이드

## 근거 데이터

한국 다문화 가정 출신국 비중 (2024년 기준, 교육부):

| 순위 | 국가 | 언어 | 비중 |
|------|------|------|------|
| 1 | 베트남 | 베트남어 | 약 30% |
| 2 | 중국 | 중국어(간체) | 약 25% |
| 3 | 필리핀 | 영어/타갈로그어 | 약 10% |
| 4 | 일본 | 일본어 | 약 8% |
| 5 | 기타 | 영어(공통) | 나머지 |

---

## 적용 우선순위

### Phase 1 — 즉시 (가장 중요)
**베트남어** (Tiếng Việt)
- 다문화 가정 중 가장 높은 비중
- 한국어와 언어 구조가 달라 정보 접근이 특히 어려움

### Phase 2 — 다음 단계
**중국어 간체** (简体中文)
- 두 번째 높은 비중
- 번체(대만)보다 간체 우선 (중국 본토 출신 비중 높음)

### Phase 3 — 장기 목표
**영어** (English)
- 필리핀, 기타 국적 공통 소통 언어
- 한국어를 배우는 중인 부모에게 영어가 더 편한 경우 포함

### 추가 검토 언어
- **일본어**: 일본 출신 비중 고려
- **타갈로그어**: 필리핀 커뮤니티 대상

- [ ] 112 (경찰) 설명

- [ ] 1366 (여성긴급전화) 설명
- [ ] 1388 (청소년전화) 설명
